Java

Java es un lenguaje de programación de propósito general, concurrente, orientado a objetos y basado en clases. Fue diseñado para tener la menor cantidad de dependencias de implementación posibles, lo que permite que las aplicaciones escritas en Java puedan ejecutarse en cualquier sistema que cuente con una Máquina Virtual de Java (JVM) compatible. Fue desarrollado por James Gosling y su equipo en Sun Microsystems (adquirida posteriormente por Oracle Corporation) y lanzado por primera vez en 1995.
